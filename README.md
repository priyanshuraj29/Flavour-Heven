#Authentication Flow (OTP)

1. User enters their mobile number/email.
2. An OTP is generated and validated.
3. On successful verification, the user is logged in.
4. Session is maintained on the client side.



# AI Chatbot

* Basic rule-based AI chatbot.
* Handles common user queries.
* Improves user engagement.
* Implemented using pure JavaScript logic.
